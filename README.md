# M&J Stitches — Demo Website

A demo frontend website built for M&J Stitches, a bespoke menswear brand based in Port Harcourt, Rivers State, Nigeria.

## Overview

M&J Stitches specializes in made-to-order menswear — suits, senators, kaftans, and more. This site was built to give the brand a proper online presence, showcase their work, and make it easy for customers to get in touch.

## Pages

- **Home** — Hero section, latest works carousel, and about section
- **Collections** — Full catalogue of clothing categories
- **Collection Detail** — Individual style page with contact form linked to WhatsApp

## Tech Stack

- React
- Vite
- Tailwind CSS v4
- Swiper.js
- React Router DOM

## Features

- Fully responsive across mobile and desktop
- Black and gold luxury aesthetic matching the brand identity
- WhatsApp-based contact flow — no backend required
- Smooth hover and transition effects throughout
- Custom Swiper carousel with manual navigation controls

## Status

Demo build. Final version will include real photography and business content supplied by the client.
